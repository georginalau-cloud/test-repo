# HEARTBEAT - 金融市场重大事件监控

## Mission

每小时推送国际和国内重大事件，延展分析其与金融市场的关联，尤其是A股市场。

## Scope

| Field | Value |
|------|-------|
| Timezone | Asia/Shanghai |
| Active Hours | 08:00-16:00 (A股集合竞价至收盘) |
| Quiet Hours Behavior | 始终推送，即使无重大事件 |
| Cadence | 每小时一次 |

## Source Mix Rules

1. **国内优先**：新华社、央视新闻、财联社、同花顺、国家统计局
2. **国际参考**：彭博、路透、BBC、CNN、X(国际账号)
3. **交叉验证**：至少一个社交媒体 + 一个权威信源
4. **时效要求**：事件发生≤6小时
5. **排除旧闻**：已充分发酵超过24小时的不报

## Scoring Model (0-100)

| 维度 | 分值 | 说明 |
|------|------|------|
| 影响范围 | 0-25 | 全球/全国/行业/区域 |
| 市场相关性 | 0-25 | 与A股/港股/美股的关联度 |
| 时效性 | 0-20 | 越新越高，>6h开始衰减 |
| 意外性 | 0-15 | 预期外 vs 预期内 |
| 用户持仓相关 | 0-15 | 关联用户持仓板块 |

## 输出格式

**每次心跳必推**，即使无重大事件也要发送简报。

**格式**：
```
📊 [日期] 市场心跳

🚨 重大事件（如有）：
- [事件列表，无则写"今日无重大事件"]

📈 持仓跟踪：
| 标的 | 代码 | 涨跌幅 | 当前浮盈 |
|------|------|--------|----------|
| [n个持仓] | ... | ... | ... |

💰 建议：无/关注/调仓
```

## 金融市场关联框架

### A股敏感板块映射
| 事件类型 | 敏感板块 |
|----------|----------|
| 央行降准降息 | 银行、地产、券商 |
| 产业政策出台 | 新能源、半导体、医药 |
| 地缘冲突 | 黄金、军工、油气 |
| 美股大跌 | A股开盘承压 |
| 汇率波动 | 出口型制造业 |

### 风险信号（需立即推送）
- 政策突发叫停
- 大面积监管调查
- 国际制裁升级
- 疫情/灾害对供应链冲击
- 央行/证监会紧急发布会

## Escalation & Cooldown

- 每 topic 60分钟冷却
- 连续2次确认才升级（紧急除外）
- 阈值触发后增加≥15分才升级
