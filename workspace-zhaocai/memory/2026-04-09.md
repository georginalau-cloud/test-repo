## 重大事件

### 数据接口故障排查
- **现象**：AkShare 调东方财富接口失败（RemoteDisconnected），Tushare 也连不上
- **根因**：东方财富对 Python requests 库强制反爬（curl 能通，Python 被掐断）
- **解决方案**：改用新浪财经接口 `https://hq.sinajs.cn/list=` + GBK解码，**稳定可用**
- **关键细节**：
  - 新浪接口返回价格单位**直接是元**，不需要二次转换
  - ETF（如516650、159566、161725）价格本身是元为单位的，≠ 股票的分
  - 之前 AkShare 统一除100 对股票正确，但对ETF错误（ETF本身价格就小）
- **脚本路径**：`scripts/stock_data.py`
- **教训**：发现"被墙"≠真被墙，先用 curl 单独验证来源
