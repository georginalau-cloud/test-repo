### 基金穿透收益计算逻辑（用户明确要求）
用户需要的不是"整个基金的NAV涨跌"，而是：
1. **穿透到个股**：每只基金里各重仓股占用用户多少资金 × 该股今日涨跌 = 用户在该股上的盈亏
2. **汇总到每只基金**：基金穿透股盈亏合计 ÷ 基金股票仓位比例 = 基金穿透贡献
3. **最终输出两张表**：
   - 表1：每只基金 → 穿透重仓股 → 占用资金 → 今日涨跌 → 盈亏金额
   - 表2：每只穿透重仓股 → 被哪些基金持有 → 总占用资金 → 总盈亏（看个股对整体的影响）
