# HEARTBEAT - 招财持仓监控

每次 heartbeat 做一件事：**快速扫描持仓异动**。
完整的金融事件分析由 cron 负责（每天 08/12/18/22 点），heartbeat 只做轻量实时扫描。

---

## ⚠️ 非交易日守卫

**先确认今天是否为A股交易日。** 五一/国庆/春节等节假日（非交易日）：直接回复 `HEARTBEAT_OK`，不做任何分析，不发任何消息。

可用以下方式判断：
- 通过 `akshare` 的 `ak.tool_trade_date_hist_sina()` 查询当年交易日历
- 或检查今日是否为周六/周日
- 非交易日直接退出，不输出任何内容

## 输出规则

**⚠️ 推送原则：报告直接发，不发确认询问。**
- 有异动 → 直接发推送报告，**不要**先发"重大事件确认，是否整理报告"之类的中间态消息
- 结论式推送，不要询问式推送

**有异动**：发送简洁推送给用户（飞书 open_id: ou_e47f62cad7fec339fba10ae6e1f5a3c9，accountId: zhaocai）

格式：
```
📡 持仓快扫 HH:MM

🔴/🟢 [标的名] [代码] [涨跌幅] [当前价]
（如有多条异动一并列出）

💱 外汇/大宗（如有异动）：
- [品种] [涨跌幅]
```

**无异动**：静默，只回复 HEARTBEAT_OK，不发任何消息。
