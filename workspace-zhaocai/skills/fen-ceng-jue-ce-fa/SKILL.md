# 分层决策法

六层决策框架用于A股智能分析，整合了实时行情、技术指标、基本面评分和风控。

## 核心脚本

执行脚本：`scripts/aura_stock_analyzer.py`

```bash
# 查看大盘
python3 scripts/aura_stock_analyzer.py --index

# 分析个股（简化版）
python3 scripts/aura_stock_analyzer.py 600519

# 分析个股（完整版）
python3 aura_stock_analyzer.py 600519 --style full
```

## 六层框架

1. **数据获取** - 实时行情 + 历史数据
2. **三层九维评分** - 价值层(40%)、交易层(35%)、风险层(25%)
3. **决策闸门** - BUY(≥75分)、WATCH(≥65分)、HOLD(≥55分)、SELL(<55分)
4. **证据分层** - A类(实时+历史)、B类(仅实时)、C类(仅推测)
5. **风控** - 2%止损线
6. **输出** - 格式化报告

## 评分规则

- **价值层**：基本面评分（默认75，可配置）
- **交易层**：技术面评分，默认65。涨幅>3%则+10，<-3%则-10；RSI>70则-10，RSI<30则+10
- **风险层**：风险评分，默认70

## 决策闸门

| 综合分 | MA20 | RSI | 动作 | 仓位 |
|--------|------|-----|------|------|
| ≥75    | 站上 | 30-70 | BUY | 30-50% |
| ≥65    | 站上 | 任意 | WATCH | 10-30% |
| ≥55    | 任意 | 任意 | HOLD | 0-10% |
| <55    | 任意 | 任意 | SELL | 0% |
