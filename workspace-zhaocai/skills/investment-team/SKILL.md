# investment-team

沛柔专属投资团队调度技能。通过召唤不同的"队员"subagent协同作战，结果汇总后输出。

## 团队成员

| 角色 | 触发指令 | 职责 |
|------|----------|------|
| 🛡️ 风控官 | `召唤风控官` | 持仓风险扫描、VaR计算、止损预警 |
| 📡 情报员 | `召唤情报员` | 实时监控国内外政策/地缘/宏观事件 |
| 📊 统计员 | `召唤统计员` | 波动率/相关性/回归/夏普比率计算 |
| 🎯 A股猎手 | `召唤A股猎手` | 选股分析、板块轮动、买卖点建议 |
| 🏛️ 港股观察员 | `召唤港股观察员` | 港股趋势跟踪、H股溢价分析 |
| 💱 外汇套利手 | `召唤外汇套利手` | 汇率风险对冲、美元/日元/欧元走势 |
| 📈 基金买手 | `召唤基金买手` | 基金筛选、配置建议、业绩归因 |

## 调度流程

1. **接收任务** → 判断需要哪个/哪些队员
2. **更新作战记录** → 飞书文档（BiqCdUKkMoxSIwxLKPTc3wtnnYe）
3. **Spawn subagent** → 携带角色prompt模板 + 持仓数据（来自MEMORY.md）
4. **汇总结果** → 整合各队员输出，整合后告知沛柔
5. **更新作战记录** → 写入执行结果

## 作战室

文档：`BiqCdUKkMoxSIwxLKPTc3wtnnYe`
表格：每次任务新建一行，格式：[时间] [召唤者] [任务描述] [状态] [结果摘要]
