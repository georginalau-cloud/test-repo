# 基金持仓穿透估算 skill

## 核心方法

每次计算基金今日预估盈亏，必须按以下步骤进行：

### 第一步：获取持仓明细 + 资产配置比例

**季报持仓明细**（前10大重仓股）：
```
URL: https://fundf10.eastmoney.com/FundArchivesDatas.aspx?type=jjcc&code={FUND_CODE}&topline=10&year={YEAR}&month={MONTH}
```

**资产配置比例**（股票/债券/现金等占比）：
```
URL: https://fundf10.eastmoney.com/zcpz_{FUND_CODE}.html
```
- 需获取"股票占净值比例"（%），用于精确穿透算法

- 自动抓取持仓脚本: `scripts/fetch_fund_holdings.py`
- 输出文件: `references/fund_holdings_realtime.json`
-持仓字段说明：
  - `占净值比例` = 该股票市值 / 基金净资产（%）
  - `持股数（万股）` = 股票数量（万股）
  - `持仓市值（万元）` = 持仓市值（万元）

### 第二步：获取今日实时股价
```python
all_stocks = ak.stock_zh_a_spot()
stocks_dict = {row['名称']: {'price': float(row['最新价']), 'chg': float(row['涨跌幅'])}
```

### 第三步：获取基金净值（上一交易日）
```python
df = ak.fund_open_fund_info_em(symbol=FUND_CODE)
latest = df.sort_values('净值日期', ascending=False).iloc[0]
nav = float(latest['单位净值'])
growth = float(latest['日增长率'])  # 上一交易日涨跌幅
```
净值日期应为最近一个交易日（非当日）

### 第四步：计算公式（两种方法）

**用户持有市值：**
```
用户持有份额 × 基金净值 = 用户持有市值（元）
```

**方法A：穿透法（个股 × 权重）**

适用场景：已知"股票配置比例"时使用此法，更精确。

```
基金今日净值变化 = Σ [ (今日个股涨跌幅 / 100) × (占净值比例 / 100) × 股票配置比例 ]

个股穿透占用资金 = 用户持有市值 × (占净值比例 / 100) × 股票配置比例
个股预估盈亏 = 穿透占用资金 × (今日涨跌幅 / 100)

估算今日净值 = 上一交易日净值 × (1 + 基金今日净值变化)
用户今日盈亏 = (估算今日净值 - 上一交易日净值) × 用户持有份额
```

**方法B：简化穿透法**

若不知道股票配置比例，可用简化版：
```
基金今日净值变化 = Σ [ (今日个股涨跌幅 / 100) × (占净值比例 / 100) ]
估算今日净值 = 上一交易日净值 × (1 + 基金今日净值变化)
用户今日盈亏 = (估算今日净值 - 上一交易日净值) × 用户持有份额
```

**净值法预估盈亏（官方，最准确）：**
```
净值法盈亏 = 用户持有市值 × (上一交易日涨跌幅 / 100)
```
注：净值法反映的是"上一交易日"的基金净值变化，非今日

**两种穿透法对比：**
- 方法A（推荐）：股票配置比例 × 占净值比例 × 今日涨跌 → 更精确，考虑了债券/现金等非股票资产
- 方法B：仅用占净值比例 → 忽略了基金中非股票部分的稳定收益

### 第五步：输出格式

每次报告必须包含：
1. 每只基金：名称、持有市值、成本、浮盈亏、净值法预估、穿透法预估、重仓股明细
2. ETF：实时价格 × 今日涨跌幅
3. 汇总：总市值、总成本、总浮盈亏、净值法总预估、穿透法总预估
4. 穿透法亮点：今日贡献最大的重仓股 TOP5

### 关键说明

- **净值法 vs 穿透法**：两者含义不同，不可混淆
  - 净值法 = 持有市值 × 上一交易日涨跌幅（官方，最准确）
  - 穿透法 = 持有市值 × 持仓占比 × 今日盘中涨跌（估算，供参考）
- 差异原因：持仓可能已调仓、非重仓股涨跌未计入
- 港股/美股基金（易方达亚洲精选等）：无A股实时数据，穿透法用净值法代替
- **数据必须真实**：禁止用估算/假设数据代替真实季报持仓
