# 异动阈值配置

## 个股异动阈值

| 指标 | 阈值 | 说明 |
|------|------|------|
| 涨跌触发 | ±3% | 单日涨跌幅超此值即触发 |
| 强势涨停 | +9.5%+ | 接近涨停 |
| 强势跌停 | -9.5%- | 接近跌停 |
| 成交量突增 | +50% | 较前一交易日放量超此比例 |
| 成交量萎缩 | -30% | 较前一交易日缩量超此比例 |

## 板块异动阈值

| 指标 | 阈值 | 说明 |
|------|------|------|
| 板块联动触发 | ≥3只同板块股票异动±3%+ | 板块集体行动 |
| 板块内涨停家数 | ≥5只 | 板块情绪高潮 |
| 板块成交额增幅 | +40% | 板块整体放量 |

## 大盘异动阈值

|------|------|------|

## 异动评分模型

```
个股异动得分 = 涨跌幅度绝对值×20 + 成交量增幅×10 + 板块联动加分×15

板块联动加分：
- 板块内3-5只异动：+15分
- 板块内6-10只异动：+30分
- 板块内10只+异动：+50分

政策/地缘突发：直接+60分（跳级推送）
```

## 推送等级

| 得分 | 等级 | 推送方式 |
|------|------|---------|
| ≥60 | 🔴 紧急 | 立即推送，声音+震动 |
| 40-59 | 🟠 预警 | 立即推送 |
| 25-39 | 🟡 关注 | 推送（可合并） |
| <25 | 🟢 记录 | 不推送，静默记录 |

## 静默规则

- **静默时段**：23:00 - 08:00（本地时间）
- **例外**：得分 ≥ 60 的紧急异动不受静默限制
- **去重冷却**：同一标的 30 分钟内不重复推送（除非异动加剧超 5%）
