# market-alert · 市场异动监控

发现持仓相关异动 → 立即飞书推送预警。

## 监控范围

### 个人持仓（9只A股）
- 300896 爱美客、600309 万华化学、600352 浙江龙盛、688363 华熙生物
- 002176 江特电机、002614 澳洋健康、000652 泰达股份
- 002551 尚荣医疗、600221 海航控股

### ETF持仓（3只场内）
- 516650 有色金属ETF、159566 储能电池ETF、161725 招商中证白酒

### 基金穿透持仓（每周一自动更新）
- 14只场外基金十大重仓股（共94只，去重）
- 高度重叠股（≥2只基金持有）自动标注重复风险
- 数据来源：`references/fund_holdings.json`（每周一8:00自动更新）
- ⚠️ 基金穿透股的异动预警（📁来源）说明：这些是你持有的基金的重仓股，其涨跌会影响基金净值。如不需此类预警，可在 cron 任务中加 `--triggers stock,etf,index` 参数排除

## 异动检测维度

| 维度 | 触发键 | 阈值 | 说明 |
|------|--------|------|------|
| 持仓个股异动 | `stock` | ±3% | 个人持仓A股 |
| 基金穿透异动 | `stock` | ±3% | 基金重仓股，标注重叠风险 |
| ETF异动 | `etf` | ±3% | 3只场内ETF |
| 大盘指数波动 | `index` | ±1% | 沪深300/深证成指/创业板指 |
| 成交量突变 | `volume` | +50% | 持仓标的较昨日放量 |

## 扫描流程

### Step 1：数据加载

```
scan_alert.py 启动
    ↓
读取 references/fund_holdings.json（基金持仓）
    ↓
合并：个人持仓(9只) + 基金穿透(94只) = 合计~100只
```

### Step 2：批量获取实时行情

通过 akshare 批量拉取所有持仓股当日行情（价格、涨跌幅、成交量）。

### Step 3：异动评分

```
个股异动得分 = 涨跌幅度绝对值×20 + 成交量增幅×10

大盘指数得分 = 涨跌幅绝对值×30

≥60分：🔴 紧急（立即推送）
40-59分：🟠 预警
25-39分：🟡 关注
<25分：🟢 记录（静默）
```

### Step 4：飞书推送

发现异动立即推送，标注来源（🏠个人持仓 / 📁基金穿透）和重叠风险（⚠️高重叠股）。

## 脚本用法

```bash
# 标准扫描（含基金穿透）
python3 scripts/scan_alert.py

# 仅个人持仓
python3 scripts/scan_alert.py --no-funds

# 仅大盘波动触发
python3 scripts/scan_alert.py --triggers index

# 调整大盘阈值为±0.8%
python3 scripts/scan_alert.py --triggers index --index-threshold 0.8

# 测试模式（不推送）
python3 scripts/scan_alert.py --dry-run

# 列出所有触发条件
python3 scripts/scan_alert.py --list
```
