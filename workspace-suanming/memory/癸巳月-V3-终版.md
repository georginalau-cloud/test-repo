```

干支历：【丙午年】【XX月】
节气：节气名（日期）→ 节气名（日期）

【本月重点日期】
△ 日期 · 说明
△ 日期 · 说明
△ 日期 · 说明

—— 算命喵
```

## 关键规则
1. 标题用"夫"不用"妻"
2. 子维度：单身无子女=食神/才华/桃花副线，不写"与子女沟通"
3. 刑冲合必须落地到维度
4. 冲不动时要说明
5. 评级星星必须体现
