# 月运分析要点备忘录

## 问题记录（2026-05-05 沛柔指正）

### 问题1：标题"妻"应改为"夫"
坤造以日主为"我"，"妻"对应的是命主的丈夫，不是八字术语的"妻"。
- 正确：夫（感情/婚姻）
- 错误：妻（感情/婚姻）

### 问题2：单身无子女者的分析要有针对性
对于单身无子女的命主，分析"子"维度时不能教条地套用"子女"概念：
- 食神代表：才华、技能创作、口福、**桃花副线**（食神生财，财星代表异性缘分）
- 子女宫代表：姻缘的副线机会，而非"与子女沟通"

### 问题3：刑冲合必须落实到具体维度的影响上
不能先列刑冲合，再列五运，两者分离。而是要：
```
刑冲合 → 影响了哪个宫/哪个十神 → 在对应维度里体现
```
- 例：三合金局（巳酉丑）→ 强化酉（配偶宫+子女宫）→ 夫维度的分析里体现"配偶宫得势"
- 例：癸冲丁 → 克制偏财丁火 → 财维度里体现"偏财有损"

### 问题4："沟通"类建议只适用于有子女者
命主单身无子女，写"耐心沟通"是无意义的空话。
应改为：本月姻缘副线有波折，感情上容易有擦肩。

### 问题5：禄运分析要展开
禄=官禄=事业，不能只写一句"宜主动争取"。应展开分析：
- 事业忙碌程度
- 有无竞争压力
- 有无小人
- 贵人方位

## 月运分析标准框架（修订后）

### 输出结构（修订后）：
1. **干支基础信息**：流月干支、藏干、空亡
2. **刑冲合梳理**（先交代，再落到各维度）：
   - 三合金局（巳酉丑）→ 对夫/子的影响
   - 癸冲丁 → 对财的影响
   - 南方三会火局（巳午未）→ 对夫/禄/寿的影响
3. **五运逐项分析**（夫/财/子/禄/寿）：
   - 每个维度给出评级和展开分析
   - 注明刑冲合在该维度中的作用机制
   - 无子女者的"子"维度应从食神/桃花副线/才华角度分析
4. **重点日期**：本月关键日期及注意事项
5. **行动建议**：用神火、喜神木，本月适合/注意的事项

### 分析要点（按维度）：

**夫（感情/桃花）**
- 配偶宫状态 + 夫星强弱
- 本月桃花是否有新缘分（新缘分信号 vs 缘分较少）
- 追求者/竞争者是否出现
- 注意事项（辨别、冷静等）

**财（财运）**
- 正财/偏财分开判断
- 正财：稳定收入/事业正财
- 偏财：投机/额外收入
- 癸冲丁的直接影响

**子（食神/才华/桃花副线）**
- 命主无子女时：食神=才华/创作/口福/桃花副线
- 金局克木 → 才华有阻滞、沟通有摩擦
- 桃花副线：食神生财，财星代表年齿略小的异性缘分

**禄（事业/官禄）**
- 事业忙碌程度（高/中/低）
- 有无竞争（甲木冲庚金）
- 有无小人（午害丑、暗中使绊子）
- 内耗情况（午自刑）
- 贵人方位

**寿（健康）**
- 印星/比肩状态
- 本月健康重点关注
- 注意哪些系统

## 关于 bazi 脚本的完善建议

### 1. prompt_for_llm 的 writing_prompt 需要加入命主状态变量

当前的 `_build_monthly_prompt` 中的 `writing_prompt` 是硬编码的，没有根据命主的婚姻状态、子女状态做分支。

建议在 `prompt_for_llm` 的 `system_context` 中增加字段：
- `is_single`: bool
- `has_children`: bool

然后在 `writing_prompt` 中根据这些字段做条件判断：
- 若 `is_single=True`："子"维度从食神/桃花副线角度分析，不写"耐心沟通"
- 若 `has_children=False`：不写与子女相关的建议

### 2. wuyu 分析结果的维度名称

当前 `wuyu_analyzer` 的输出使用"妻财子禄寿"作为键名，但坤造应改为"夫财子禄寿"。

建议在 `wuyu_analyzer` 的输出结果中增加一个 `is_male: bool` 标志，或在输出键名中根据性别做转换。

### 3. 刑冲合的解读要落地到具体宫位

当前 `interactions` 输出的是"泛化的影响"（如"子宫五行大旺，影响显著"），没有具体说明影响的维度。

建议在 `analyze_zhi_relations` 的结果中增加维度标注，如：
```python
{
  "type": "三合",
  "members": ["巳","酉","丑"],
  "effect_dimensions": ["夫", "禄", "寿"],  # 哪些维度受益
  "negative_dimensions": ["子"],  # 哪些维度需防
  ...
}
```

## 日运分析要点（2026-05-10 补录）

### 十神关系（每次必须核查）
- 甲木对癸水日主：**食神**（癸水生甲木，异性相生）
- 己土对癸水日主：**七杀**（己土克癸水，官杀克日主）
- 庚金对癸水日主：**正印**（庚金生癸水，异性相生）
- 丁火对癸水日主：**偏财**（丁火被癸水克）

### 忌神判断
- 原局忌神是**土**，不是金
- 金是印星（生水之物），对日主有情
- 金旺泄火气，对用神火而言是消耗，但不是命局忌神

### 自刑条件
- 必须同一地支出现**2次以上**
- 午出现1次 → 午午自刑不成立
- 昨天壬午日：流年丙午+流日壬午各含1个午 → 午午自刑成立

### 合会局取地支原则
- 要取**所有层级**的地支：原局四柱 + 流年 + 流月 + 流日 + 时柱
- 时柱己未是三会火局（巳午未）成立的关键

### 天干冲的正确理解
- 甲庚冲 = **食神冲正印**（不是竞争）
- 食神冲正印代表：学习/证书/文书/学历方面有摩擦阻滞
- 竞争/小人应看七杀、比劫

### 食神代表的含义
- 才华创作、表达欲、口福、桃花副线（命主无子女时）
- 竞争/小人不是食神代表的内容

### 命局忌神 vs 用神相关五行
- 金是忌神（土）所生，但金本身不是忌神
- 三合金局对用神火的人而言是"忌神旺"，因为金泄火气
- 但这不是说金是命局忌神，而是说金旺对火用神构成消耗

### 日运分析的常见错误
1. 把天干食神说成"地支的藏干"
2. 把自刑当成"单个地支就能成立"
3. 把金定性为命局忌神
4. 把甲庚冲解读为"竞争"（应为食神冲正印）
5. 甲己合 = 食神合七杀（不是"财星合身"）
6. 未是时支本气，不是年支本气
### 颜色忌讳分析的正确顺序
**每次分析颜色/方位前，必须先确认：用神、喜神、忌神**
1. 确认用神（核心用五行）
2. 确认喜神（辅助五行）
3. 确认忌神（需要规避的五行）
4. 再开始分析颜色宜忌

错误顺序：先入为主说"忌XX色"，然后套到用神喜神上

### 案例：2026-05-13
- 原局用神：火
- 原局喜神：木
- 原局忌神：土
- 本喵错误说"忌绿色"——把喜神木说成忌讳
- 正确：喜神木（绿色）今天应该用，木生火，帮助用神
- 教训：分析颜色前必须先确认用神喜神忌神，这是最基本的步骤，不能跳过
