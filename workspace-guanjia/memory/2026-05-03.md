1. **美伊谈判陷入僵局**
   伊朗提交了14点谈判方案，特朗普说"难以接受"，鲁比奥随后访问梵蒂冈，表面是外交行程，实际是去解释美国在中东的立场。霍尔木兹海峡是焦点——全球最大集装箱航运公司已经开辟新航线绕道，说明市场已经在定价风险。油盟+宣布增产，但增产配额这种事历来是说的多做的少。霍尔木兹如果真的封锁，化肥、液化气、化工品全受影响，不只是油价的问题。

2. **特朗普与教宗良十四世互怼**
   这事本身就够荒诞——一个美国前总统/现总统候选人和一个罗马天主教皇打口水战。背景是教宗呼吁停火、关注加沙人道危机，特朗普反驳。教宗在这个位置上，说什么都有外交分量，但换来的往往也是更大的反弹。这场口水战背后是宗教、道义、地缘政治的混杂，鲁比奥去罗马估计就是为了降温。

3. **俄乌战争继续无人机互袭**
   双方每天都在用无人机互相打击，平民伤亡报告不断。这场战争正在变成一场消耗战，没有奇迹，只有钝刀割肉。

4. **日本实施重新雇佣制**
   65岁退休后可以以同薪续聘，听起来是对老年人的保障，但企业愿不愿意真金白银地掏这个钱是两回事。少子化＋老龄化，日本的劳动力结构压力已经到了倒逼制度改革的临界点。这件事比表面上看起来要严肃得多。
