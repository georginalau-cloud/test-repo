## 三、今天国内外大事

### 1. 魏凤和案一审宣判

前国防部长魏凤和因受贿罪被判处死刑缓期二年执行，缓期满后减为无期徒刑，终身监禁，不得减刑假释。同一天李尚福案也宣判了。两个军队系统的高级官员同日出判决，信号很明确：反腐没有休止符，不管位置多高。

### 2. 美伊霍尔木兹海峡：局势微妙，协议在望

伊朗船试图突破封锁，美国战机开火。但同时，14项谅解备忘录正在审议，美伊接近达成协议——暂停铀浓缩、解除制裁、重开霍尔木兹。这个剧本很有意思：一边打一边谈，打是为了在谈判桌上多拿筹码。霍尔木兹海峡是全球能源咽喉，一旦真的恢复正常，油价大概率要抖一抖。

### 3. SpaceX IPO 文件披露：马斯克拿稳绝对控制权

SpaceX 上市在即，估值约 1.75 万亿美元，创历史纪录。文件显示马斯克将拥有几乎不受制约的决策权，投资者被禁止起诉。这不是普通的公司治理问题——当一家控制全球大半卫星互联网、NASA 核心供应商、同时运营 Starlink 和 Starship 的公司上市，还附带这样的治理结构，那这不只是商业新闻，是地缘政治级别的议题。

### 4. 中国对非洲零关税生效（5月1日）

53个建交非洲国家全部进口商品零关税，33个最不发达国家已生效，另有20个最不发达国家享受两年期零关税。这步棋走得比较长远，布局非洲供应链和外交影响力，美元债务和"小院高墙"之外在铺另一条路。
