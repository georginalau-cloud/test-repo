# 机票监控方案 - 首尔/曼谷

## 监控需求
- **出发地**: 上海 (SHA/PVG)
- **目的地**: 首尔 (ICN/GMP) 或 曼谷 (BKK/DMK)
- **时间**: 清明、五一、近期周末
- **行程**: 3天往返 (可错峰1天)
- **目标**: 便宜往返机票

## 关键日期

### 2026年节假日
1. **清明节**: 4月4日-6日放假 (4月3日可错峰)
2. **劳动节**: 5月1日-3日放假 (4月30日/5月4日可错峰)

### 推荐出行日期组合

#### 首尔 (3天2晚)
1. **清明错峰**: 4月3日(四)-4月5日(六) 或 4月6日(日)-4月8日(二)
2. **五一错峰**: 4月30日(四)-5月2日(六) 或 5月3日(日)-5月5日(二)
3. **周末出行**: 任意周五晚-周日晚

#### 曼谷 (3天2晚)
1. **清明错峰**: 4月3日-5日 或 4月6日-8日
2. **五一错峰**: 4月30日-5月2日 或 5月3日-5日
3. **周末出行**: 周五晚-周日晚

## 监控策略

### 1. 价格监控点
- **首尔**: 正常价 ¥1500-2500，特价 ¥800-1500
- **曼谷**: 正常价 ¥2000-3000，特价 ¥1200-2000
- **目标价**: 低于正常价30%时提醒

### 2. 航空公司关注
- **首尔航线**: 春秋、吉祥、东航、韩亚、大韩
- **曼谷航线**: 春秋、吉祥、东航、泰航、亚航

### 3. 最佳预订时间
- 提前2-4周监控
- 周二/周三价格较低
- 凌晨时段可能有特价

## 自动化监控方案

### 方案A: 使用旅行比价网站API
1. Skyscanner API (需要注册)
2. 天巡价格提醒
3. 携程/飞猪价格监控

### 方案B: 网页监控脚本
1. 定期抓取比价网站
2. 设置价格阈值
3. 价格达标时发送提醒

### 方案C: 手动定期检查
1. 每周检查2次
2. 关注航司促销邮件
3. 使用比价App设置提醒

## 实施建议

### 短期方案 (手动)
1. 每周二、四检查价格
2. 关注航司官网促销
