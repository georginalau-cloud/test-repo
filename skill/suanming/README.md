# 八字精批分析系统

完整的八字命盘分析和五行运势预测工具。

## 整体结构
suanming-bazi-analyzer/
├── SKILL.md
├── README.md
├── .gitignore
├── bin/
│   └── bazi_detailed         ————这是唯一对外入口。（负责：接收用户输入；统一参数格式；调起后续主流程；返回最终结果）
├── src/
│   ├── cities_longitude.py   ————真太阳时工具模块。（负责：城市经度查询；均时差计算；真太阳时修正；真太阳时&平太阳时校准）
│   ├── yuanju.py             ————原局排盘模块。（负责：接收真太阳时后的出生信息；计算四柱原局；输出原局结构；提供原局分析所需的基础数据）
│   ├── dayun.py              ————时间展开模块。（负责：顺逆排；起运计算；大运序列；流年序列；流月序列；流日序列）
│   ├── tech_analyzer.py      ————统一分析调度模块。（负责：串联并分析不同层级命盘内容：生克路线；刑冲合会害破；格局；十神与六亲；神煞；学术性 comments）
│   ├── analyzer_hub.py       ————结构化八字分析包装器。（把排盘结果整理成标准 JSON；兼容旧调用方式；做排盘和分析之间的数据衔接）————彻底调通后可停用
│   └── bazi_engine/          ————这一层是传统命理原始资料和兼容脚本层。为“底层资料库 + 历史 CLI 兼容区”。
│       ├── bazi.py               ————原始八字 CLI 脚本。（负责：原始文本排盘；大量命理注释输出；历史兼容）。最重的底层脚本之一。
│       ├── common.py             ————通用辅助函数。（负责：空亡判断；合冲判断；拱合判断；阴阳判断）
│       ├── datas.py              ————命理数据字典总表。（负责保存：十神映射；神煞表；格局表；纳音；长生十二运；口诀、经典短句；其他基础数据）
│       ├── ganzhi.py             ————干支基础定义与工具。（负责：天干地支列表；干支映射关系；六合六冲三合三会等基础表；阴阳、五行、关系定义）
│       ├── luohou.py             ————罗喉日/时与风水辅助脚本。（负责：罗喉日时判断；风水择日相关提示；九宫飞星等辅助输出）
│       ├── shengxiao.py          ————生肖合婚辅助脚本。（负责：生肖六合；三合；三会；冲刑害破）
│       ├── sizi.py               ————三命通会类摘要。（负责：保存经典短评；提供部分命理文本引用；作为分析时的参考资料来源）
│       └── yue.py                ————穷通宝鉴月份参考。（负责：按月份提供命理参考；保存月令分析文本；作为格局/调候判断的辅助材料）
└── lib/
    └── five_yun_analyzer.py  ————五运分析核心库。（负责：妻财子禄寿；原局五运分析；当前大运五运分析；未来可扩展到流年/流月/流日五运分析）


## 功能特性

### 1. 八字排盘
- 真太阳时调整（基于城市经度）
- 干支推导
- 纳音五行
- 十神判断

### 2. 命盘分析
- 日主强弱判断
- 用神忌神分析
- 格局分类
- 特殊格局识别（魁罡格、从格等）

### 3. 大运分析
- 大运排序
- 大运干支推导
- 大运吉凶评估
- 流年预测

### 4. 五行分析
- 五行分布统计
- 五行强弱对比
- 调候用神建议
- 五运深度分析

### 5. 命理学参考
- 《穷通宝鉴》分析
- 《三命通会》参考
- 十二时辰解读
- 星宿吉凶评估
