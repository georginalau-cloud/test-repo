# 八字精批分析系统

完整的八字命盘分析和五行运势预测工具。

## 功能特性

### 1. 八字排盘
- 真太阳时调整（基于城市经度）
- 干支推导
- 纳音五行
- 十神判断

### 2. 命盘分析
- 日主强弱判断
- 用神忌神分析
- 格局分类
- 特殊格局识别（魁罡格、从格等）

### 3. 大运分析
- 大运排序
- 大运干支推导
- 大运吉凶评估
- 流年预测

### 4. 五行分析
- 五行分布统计
- 五行强弱对比
- 调候用神建议
- 五运深度分析

### 5. 命理学参考
- 《穷通宝鉴》分析
- 《三命通会》参考
- 十二时辰解读
- 星宿吉凶评估
